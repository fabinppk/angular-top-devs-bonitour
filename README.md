# Angular with SSR + SEO

Esse projeto tem o intuito de usar Angular com server-side-render.

## Development server

`$ npm install`

`$ npm run start`

Acesse `http://localhost:4200/`

## Build

`$ npm install`

`$ npm run build`

## SSR Build

`$ npm install`

`$ npm run build:ssr`

## Production server

`$ npm install`

`$ npm run build:ssr`

`$ npm run serve:ssr`

Acesse `http://localhost:4000/`
